Jadi sistemnya ini, alurnya: pertama, admin LPPM dapat melakukan login dengan menginputkan email dan password. Kemudian setelah login, admin menuju ke halaman input data peserta. Di situ, admin LPPM bisa mengimport data CSV (data peserta = data mentah) yang didapat dari google form yang dibuat sendiri oleh admin LPPM. Tapi sebelum mengimport, Admin LPPM harus terlebih dahulu menambah atau membuat kelompok-kelompoknya dulu agar dapat nomor kelompok di halaman buat kelompok KKN (halaman baru). Nantinya kelompok-kelompok yang sudah dibuat oleh admin LPPM itu akan tampil di halaman input data peserta di dalam tabel kelompok. Kalau sudah di buat manual satu-satu sampai jumlah yang ditentukan, LPPM mengimport data CSV (data peserta) tadi ke dalam halaman input data peserta berupa tombol choose file. Kemudian, admin menekan tombol submit. Maka data yang ada di tabel kelompok nya di situ akan terisi sendiri. Kemudian, Admin LPPM menekan tombol generate, maka sistem akan melakukan proses randomisasi secara otomatis menggunakan metode constraint-based randomization dan weighted random distribution untuk membagi kelompok KKN berdasarkan kriteria dan bobot, yaitu ada ipk, asal wilayah, gender, agama, dan riwayat penyakit. Jika sudah dirandomisasi, maka sistem akan menampilkan susunan hasil pembagian kelompok KKN di dalam tabel kelompok tersebut (sudah jadi data susunan kelompok, data peserta --> data susunan kelompok). Kemudian jika ada kendala seperti permintaan khusus dari peserta, maka admin LPPM juga dapat CRUD Kelompok, seperti memindahkan dan menukar peserta, menghapus kelompok, dan yang terpenting dapat membagi DPL dan APL ke kelompok KKN di halaman input data peserta disini. Admin juga dapat mengedit dan menghapus data peserta di halaman ini. Kemudian admin LPPM dapat menginput data DPL dan APL di halaman input data DPL dan APL (Berbeda halaman).  Setelah data DPL dan APL keduanya diinputkan, maka sistem akan mengupdate tabel kelompok yang ada di halaman input data peserta setelah diinputkan data DPL dan APL tadi. Di situ, admin LPPM bisa menekan tombol tukar dan pindah peserta ke kelompok lain yang sudah disediakan. Sistem akan memunculkan pop up dialog box untuk proses penginputan oleh admin LPPM sendiri. Setiap proses atau aktivitas yang dilakukan seperti mengedit atau mengelola data susunan kelompok, akan otomatis tersimpan. Admin LPPM dapat melihat Riwayat Aktivitas Log di menu Riwayat aktivitas. Jadi jika susunan kelompok sudah fix, maka admin LPPM hanya perlu menekan tombol save as PDF untuk menyimpan arsip dokumen tersebut atau admin bisa mencatat di excel atau di dokumen untuk kemudian di bagikan ke peserta KKN, DPL, dan APL.
